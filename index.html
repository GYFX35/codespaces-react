Cybersecurity Tools Collection

This repository contains a collection of various security-related tools. Click on a tool's documentation to learn more about its functionality and how to use it.

  • Account Pwned Checker

    A command-line tool to check if an email account has been compromised in a known data breach using the 'Have I Been Pwned?' API.

    View Documentation

  • Sensitive Data Scanner

    A tool to recursively scan a directory for files containing sensitive data patterns like API keys, private keys, and credit card numbers.

    View Documentation

  • Port Scanner

    A simple, multi-threaded TCP port scanner to find open ports on a target host.

    View Documentation

  • IoT Simulator

    A simulator for IoT devices.

    View Documentation

  • Network Mapper

    A tool to generate network topology diagrams from a CSV file.

    View Documentation

  • Scam Detector

    A tool to detect scams in social media.

    View Documentation

  • Simple Browser

    A simple web browser.

    View Documentation

  • Wi-Fi Analyzer

    A tool to analyze Wi-Fi networks on Linux systems.

    View Documentation
